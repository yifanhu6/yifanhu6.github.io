Introduction

This dataset is called GDP.csv, and it tells the GDP changes from 1947 to 2019, and the data was collected every quarter. I got this dataset from our course website https://uiuc-ischool-dataviz.github.io/is445AOG_fall2020/week01/data/GDP.csv I am trying to show the trend of GDP development from 1947 to 2019 in the plot. Thus, I choose the line graph to present the changes, which is a frequent way to visualize the time series data and show the trends. I set the title of x axis to be “Date” and y axis to be “GDP”, which can clearly show the relationship between these two factors. I also change the ranges of y axis to make the visualization better to be viewed. I added another two built-in components. One of it is a table and it displays all the variables and values in the dataset. Another one is an action button, which will display an alert when hitting the “click me”.
